# 구조 신박성 — 재진단 (변경: 새 시스템 불필요, 기존 장치가 무기억)

## 사용자 통찰(핵심)
"보조괴담·중간사건으로 이미 시도했는데 같은 불만이 또 나온다 = 실제로 안 돌아가고 있다."
→ 맞음. 그리고 코드를 보니 **구조 다양성은 애초에 보조괴담·중간사건의 일이 아니었다.**

## 코드로 확인된 진짜 원인
- **'게임의 종류'를 정하는 건 `world_rules 구조`다** (`ScenarioArchetypes.RULES`). 유형이 **34종**
  (A티어 26종): 정보격리형·통신유인형·비개입형·규칙목록·금기행동형·시한의식형·도덕딜레마·집단환각형·
  환경침식형·역할극강제형·인과역전형·전원괴이형·진영형·수수께끼형·감정금기형·감각동기화형·신체기생형·
  사후정보형·시간루프형·기억망각형·공간왜곡형(밀실·백룸)·차원분리협력형·천년운영형… — 각각이 ★다른 게임★.
  3+스테이지는 "구조 먼저 고정, entity를 여기 맞춰 도출"까지 명시(구조 우선 의도는 이미 있음).
- **그런데 이 34종 선택에 '기억'이 없다.** `worldRulesBlock`은 매 생성 `Collections.shuffle`로 후보를
  새로 섞고, 유일한 반복 방지가 프롬프트 문구 **"직전 방과 다른 유형으로"** 뿐이다 — (a) '직전 1개'만,
  (b) 프롬프트 힌트일 뿐 엔진 제외 없음, (c) "고르거나 창작하라"라 모델이 고확률 친숙 구조(저주+약점+의식)로
  회귀. → 스테이지 1·3·5가 같은 구조로 반복돼도 아무것도 안 막는다.
- 대조: **괴담 '이름' no-repeat는 실제 원장(`.familiar_history`)이 있어 작동한다.** 구조는 그 원장이 없다.
- **보조괴담·중간사건은 '압박(threat) 층'**이다(마감·2차 위협). 핵심 공략 루프를 바꾸지 않으니 아무리 넣어도
  체감 구조는 '메인 괴담 조사→약점→실행' 그대로 = 불만 재발의 진짜 원인.

## 결론 — 새 시스템(clear_shape/structure_motif enum)은 과설계였다
필요한 건 **이미 검증된 이름 no-repeat 원장을 world_rules 구조 축에 그대로 복제**하는 것뿐:
1. 생성 성공 시 고른 구조 유형(RULES의 어느 Arch인지 = entity.type/world_rules.core 키워드로 식별)을
   `.structure_history`에 기록(이름 원장과 같은 상위 폴더).
2. `worldRulesBlock`의 후보 셔플에서 **최근 창(캠페인 전체 또는 N)에 쓴 구조를 하드 제외**(프롬프트 힌트가
   아니라 후보 목록에서 실제로 뺀다 — 이름 no-repeat와 동일). 34종이라 5스테이지 캠페인은 여유롭게 안 겹침.
3. 캠페인(새 게임) 시작 시 구조 창 리셋(웹소설 감성 = 한 앤솔로지 안에서 안 겹침).

## 격상(점점 큰 사건) — 이미 반쯤 되어 있음
- 스케일 단조 상승(로컬→코즈믹)·DC 하한은 이미 있음. 여기에 **구조가 매 스테이지 실제로 달라지면**
  "다른 게임 + 더 큼" = 사용자가 원한 그림.
- (선택) 복합도: 후반 스테이지는 구조 2개를 얽음. 단 조기종결·해결의 격 원칙 유지(복합도=판의 얽힘,
  공략은 단일 행위). 이건 2차 — 먼저 '구조 no-repeat'만으로도 체감이 크게 달라질 것.

## 남은 검증(별개 축, 코드로 확인 필요)
- **런타임 충실도**: 고른 구조가 실제 플레이에서 살아남나? 예) 밀실탈출·규칙목록 유형이 런타임 GM에서
  '그냥 조사'로 붕괴하지 않나? 저사양 GM이 구조를 못 굴리면 생성 다양성이 무의미. → 실제 .gdam 생성물
  또는 플레이 로그로 '구조가 collapse_condition·main_events·GM 서술에 실제로 반영되는지' 확인 필요.
  (이게 두 번째 원인일 수 있다 — 생성은 구조를 넣는데 런타임이 무시.)

## 제안 실행 순서
1. **구조 no-repeat 원장**(위 1~3) — 작음, 검증된 패턴 복제. 먼저.
2. **런타임 충실도 감사** — 로그/생성물로 구조가 살아남는지 확인, 안 되면 GM 프롬프트에 '이 구조를 매
   응답에서 실제로 굴려라' 강제.
3. (선택) 복합도 상승.
