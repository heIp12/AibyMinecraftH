# 구조 다양성·복합도 — 수렴 결론 (GPT 조율 완료)

## 진단 정정 (내 초기 진단은 과했음)
- 초기: "구조 시스템이 없어 전부 단조롭다" → **과함.** GPT가 실제 13개 .gdam을 검토: 핵심 공략 구조가
  이미 다양(격리·치료·장례·자백·재앙방지·규칙거부·화해·루프탈출·다중전투). 13개 전부 side:true·combat:true
  사건 보유. **생성은 잘 작동.**
- 재진단: "world_rules 구조 no-repeat 없음" → 부분 사실이나 핵심 아님.
- **진짜 약점 = 런타임 집행(코드로 확인).** `GameStateManager.fireDueEvents`는 id·time·label·effect·
  is_end·combat·threat만 소비하고 **side·side_scale·deadline·event_role·key_clue는 안 읽는다.** 중규모
  사건은 '시작 문구 1회 + 위협 상승'만 하드코딩, 마감·성패·지속효과는 전부 GM 기억 의존 → 저품질 GM이
  몇 턴 뒤 잊으면 .gdam엔 있어도 플레이엔 없다. = "생성됐는데 체감 안 됨"의 유력 원인.

## 작동/미작동 정리 (GPT)
- 핵심 구조 다양화: ✅ · 보조괴담 생성(확률형): ✅ · side 생성: ✅ · combat 생성·소집: ✅ · 사건 시각·위협: ✅
- side 사건 마감·성패·지속효과: ❌ GM 의존(불안정) ← **핵심 구멍**
- 후반 side 개수: △ 지침보다 적게 생성(5st 2~3 목표에 1~2, 6st 3~5 목표에 1~2)
- event_role 다양성: △ 15개 중 인명희생5·지속출혈5·방해유발3에 편중, 6종(해결/탈출/등장관문·즉시파국·
  함정강화·정보대가)은 0회 → 사건이 '고장/위험→마감 못 막으면 지속피해·희생'으로 수렴
- 프롬프트 충돌: side=선택적 / 정답 알면 조기종결 / 예정사건 전부발화 — 저품질 모델에 3규칙 상충
  (사건 잊고 조기종결 or 해결된 판 지연 둘 다 가능)

## 수렴 방향 (새 시스템 추가 ❌ · 기존 복합요소가 플레이 중 안 사라지게 ✅)
1. **런타임 중규모 사건 트래커(핵심)**: side 사건에 활성 상태 + 마감 큐 + 실패 결과를 엔진이 실제 집행.
   - 사건 발화 시 active로 등록(id·deadline·event_role·effect). 매 턴 GM 문맥에 '진행 중 사건 N: 마감 HH:MM,
     미해결 시 <효과>' 주입(GM이 안 잊게). deadline 도달 && 미해결(EVENT_BLOCK 없음)이면 엔진이 event_role별
     실패 결과 적용(지속출혈=HP 지속 -, 인명희생=대상 위험, 방치강화=위협 상승 등)·로그.
   - ★조기종결 존중★: 정확한 해법으로 판이 끝나면(onClearEnding) 미해결 active 사건은 전부 폐기(사건 수
     채우려 종결 막지 않음). GPT 4번.
2. **구조 히스토리는 가볍게**: 완전 no-repeat 원장이 아니라 '직전 스테이지와 ★주 기믹이 같은지만★' 확인해
   같으면 다른 유형 유도(GPT 2번). 34종 풀이라 이 최소 게이트로 충분.
3. **생성 보강(작음)**: event_role 다양성 넛지(안 나온 6종을 후보에서 강조·최근 쓴 role 회피) + 후반 side
   개수 준수 강화(약하게 지켜짐).
4. **프롬프트 충돌 해소**: '예정 사건은 진행 중일 때만 뼈대 — 정확한 해법으로 종결되면 남은 사건은 폐기'로
   조기종결 우선 명문화(해결의 격·조기종결 원칙과 정합).

## 폐기
- 초기 제안한 clear_shape/structure_motif enum 신설 = 과설계(기존 world_rules 34종·보조괴담·side와 중복).
- 복합도 '추가'보다 '이미 생성된 복합요소의 런타임 생존'이 실제 필요.

## 리스크
- 런타임 트래커는 비동기 턴 루프(전투 소집이 이미 사는 곳)를 건드림 → 저위험부터(마감 큐·GM 주입 먼저,
  자동 실패 적용은 신중히). 자동 실패가 과하면 조기종결·정보경제와 충돌하니 event_role별 실패는 '만회 가능
  대가'로.
- 자료가 시나리오뿐 플레이 로그 없음 → 실제 발화율 미확정. 트래커 넣고 로그로 재검증 필요.
